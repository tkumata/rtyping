# ADR 202604051106

- Timestamp: 2026-04-05 11:06
- Decision: 外部 API による文字列生成は `usecase` 層へ集約し、UI 層はプロバイダ選択、設定編集、結果表示のみを担当する。
- Rationale: タイトル画面と Config 画面の責務を入力と遷移に限定し、Google AI Studio と Groq の差異を UI から分離するため。

- Timestamp: 2026-04-05 11:06
- Decision: 設定は `~/.config/rtyping/config.json` の単一 JSON ファイルへ保存し、`google` と `groq` の設定を独立して保持する。
- Rationale: プロバイダごとの URL、モデル、暗号化済み API key を一元管理し、Config 画面からの編集結果を永続化しやすくするため。

- Timestamp: 2026-04-05 11:06
- Decision: `API key` は `config.json` へ AEAD による暗号文として保存し、復号鍵は同一設定ディレクトリ内の専用鍵ファイル `config.key` に分離して保持する。
- Rationale: `config.json` 単体では復号できない構成を保ちつつ、復号時の改ざん検知も行えるようにするため。

- Timestamp: 2026-04-05 11:06
- Decision: CLI 引数 `--google` と `--groq` は起動時の生成プロバイダを選択する排他的フラグとして扱い、未指定時は既存のローカル生成を使う。
- Rationale: 既存の操作性を維持しながら、外部 API 利用を明示的な opt-in にするため。

- Timestamp: 2026-04-05 11:06
- Decision: タイトル画面は `Menu` 状態へ拡張し、`Start Game` と `Config` の選択 UI、および `Config` 状態での編集フォームを追加する。
- Rationale: 既存の `Intro -> Typing -> Result` を大きく崩さず、設定編集導線をゲーム開始前に集約するため。
