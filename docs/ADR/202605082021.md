# ADR: WPM Trend ブロック枠線色を薄い黄色にする

## Status

Accepted

## Context

WPM グラフは `src/presentation/ui/render/wpm_graph.rs` の共有描画関数から Typing 画面と Result 画面へ表示される。現状のブロック枠線色はシアンであり、ユーザーは WPM Trend ブロックの枠線だけを薄い黄色にしたい。

この変更は表示上の枠線色に限定し、WPM 履歴、グラフ本体の線色、高値部分のオレンジ強調、入力判定、画面遷移は変更しない。

## Decision

WPM グラフ用 `Block` の枠線色を共有レンダラ内で薄い黄色へ変更する。

Typing 画面の `WPM Trend` と Result 画面の `Final WPM Trend` は同じ共有レンダラを使うため、画面ごとの個別指定は追加しない。

## Consequences

- WPM Trend 系ブロックの枠線が薄い黄色で統一される。
- グラフ本体の緑系線色と高値部分のオレンジ強調は維持される。
- 画面ごとの分岐を増やさないため、表示仕様の追跡箇所は `wpm_graph.rs` に集約される。
