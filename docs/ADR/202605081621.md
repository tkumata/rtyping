# ADR: タイトルメニュー選択マーカーを `▶︎` に変更する

## Status

Accepted

## Context

起動直後のタイトル画面では、現在選択中のメニュー項目を `>` で表示している。ユーザー要求により、この選択記号を `▶︎` へ変更する。

この変更は表示上の選択マーカーだけを対象とする。メニュー項目、選択順、キーバインド、生成元選択、`Stats` / `Config` 遷移は既存動作を維持する。

## Decision

- `Menu` 画面の選択中項目は、ラベル左に `▶︎` と空白を付けて表示する。
- 非選択項目は、既存どおり空白プレフィックスでラベル位置を揃える。
- 実装箇所は `src/presentation/ui/render/menu.rs` の `menu_line` に限定する。

## Consequences

- 起動直後の `Start Game` は `▶︎ Start Game` として表示される。
- 上下キーで選択移動した場合も、現在選択中の項目だけが `▶︎` を持つ。
- 入力処理やアプリ状態には影響しない。

## Verification

- `make check`
- `make build`
