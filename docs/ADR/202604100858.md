# ADR: `typing.rs` のテストモジュール配置を是正する

- Date: 2026-04-10 08:58
- Status: Accepted

## Context

`src/presentation/ui/render/typing.rs` では `#[cfg(test)] mod tests` の後ろに `render_footer` が定義されており、`cargo clippy` で `clippy::items_after_test_module` 警告が発生している。

## Decision

`typing.rs` の通常関数をテストモジュールより前へ並べ替え、`#[cfg(test)] mod tests` をファイル末尾に配置する。

## Consequences

- 挙動を変えずに Clippy 警告を解消できる。
- 今後の描画モジュールでも、テストモジュールを末尾へ置く方針を適用しやすくなる。
