# ADR: 外部プロバイダ開始メニューの表示条件

## Context

Google AI Studio と GroqCloud は、`API URL`、`API Key`、`Model` の3項目が揃っていないと生成を開始できない。未設定の状態でもタイトル画面に外部 API 開始メニューが表示されると、選択後に設定不足エラーへ進むだけの導線になる。

## Decision

- タイトル画面の `Start Game via Google AI Studio` は、Google AI Studio の `API URL`、`API Key`、`Model` がすべて空でない場合だけ表示する。
- タイトル画面の `Start Game via GroqCloud` は、Groq の `API URL`、`API Key`、`Model` がすべて空でない場合だけ表示する。
- 完全性判定には `ProviderConfig::is_ready()` を使い、空白のみの値も未設定として扱う。
- 描画と上下キー移動は同じ表示可能メニューリストを使う。

## Consequences

- 未設定プロバイダの開始項目は選択できないため、ユーザーは先に `Config` で必要項目を入力する流れになる。
- `Start Game`、`Practice Mode`、`Stats`、`Config` は常に表示される。
- 生成処理側の設定不足エラーは、防御的な失敗処理として維持する。
