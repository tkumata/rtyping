# R-Typing Help

## CLI Options

```text
-t, --timeout <SECONDS>  Timer duration (default: 60)
-l, --level <WORDS>      Number of words (default: 30)
    --freq <FREQUENCY>   Sound frequency in Hz (default: 80.0)
-s, --sound              Enable background music
```

## Keyboard Shortcuts

### Title Screen

- **Enter**: Start typing
- **h**: Show/Hide this help
- **Up / Down**: Scroll help text (when visible)
- **Esc**: Quit (or close help)
- **Ctrl+c**: Quit

### Typing Screen

- **Backspace**: Delete last character
- **Esc**: End typing and show results
- **Ctrl+c**: Quit

### Result Screen

- **Enter**: Quit

## Tips

- Focus on accuracy over speed
- WPM = (typed chars - misses) / (5 \* time / 60)
